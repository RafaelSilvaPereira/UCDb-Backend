# Caso de uso 1:
Cadastrar usuário:
Rota = "/api/v1/students/"
Body no formato = 
"{
	"email" : "email_do_usuario",
	"firstName" : "primeiro_nome",
	"secondName" : "segundo_nome",
	"password" : "senha"
}"

Login de usuário:
Rota = "/api/v1/login/"
Body no formato =
"{
	"email" : "email_do_usuario",
	"password" : "senha"
}"
Retorna o token do usuário.
